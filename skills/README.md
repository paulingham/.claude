# Skills

Outside contributors: see `../CONTRIBUTING.md` or run `scripts/new-skill.sh`.

`_template/` is the internal maintainer pipeline-skill template. It includes
`verdict`, `phase`, and `dispatch` frontmatter fields used by pipeline skills.
If you are adding a reference or clean-code skill, use `scripts/new-skill.sh`
instead — it copies from `templates/skill-reference/SKILL.md` which has the
correct minimal frontmatter for outside contributions.
