# Claude Code Known Bad Versions

This file documents Claude Code versions with known harness compatibility issues.
The `_lib/session-start-version-check.sh` hook compares the running version against
`version-pin` and surfaces this doc when a mismatch is detected.

## Tracking Convention

Add an entry here when a specific Claude Code version causes harness issues.
Format: `## vX.Y.Z — <one-line issue summary>`. Include the date discovered.
